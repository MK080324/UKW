# 术语表

> 本文件定义全文档共用的中英术语映射。所有 Writer Agent 在 D1-D5 中使用以下术语时，必须与本表一致。
> Phase 0 由 Lead 初始化，Phase 1 由 Writer-Procedural 扩充，Phase 4 最终定稿。

## 使用规则

1. 术语首次出现时使用完整格式：`中文（英文全称, 缩写）`
2. 后续出现可直接使用中文或缩写
3. 同一文档中同一术语必须使用相同的中文翻译
4. 如有多个通行译法，以本表指定的为准
